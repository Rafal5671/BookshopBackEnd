## Description

BookshopBackEnd is the backend for an online book store, developed with **Spring Boot**. It provides secure **user authentication** with **JSON Web Tokens (JWT)**, **role-based access control** (admin and user), and integration with **PostgreSQL** for data storage. The backend also supports **payment processing via Stripe**.

This project demonstrates full-featured backend development, secure API design, role management, and payment integration.

---

## Features

* User registration and login with JWT authentication
* Admin and user roles management
* Product and book catalog management
* Shopping cart and order handling
* Stripe payment processing
* PostgreSQL database integration
* RESTful API endpoints for frontend consumption
* Secure password hashing and authentication flow

---

## Technology Stack

* **Backend Framework:** Spring Boot
* **Database:** PostgreSQL
* **Authentication:** JWT (JSON Web Tokens)
* **Payment Gateway:** Stripe
* **Language:** Java
* **Security:** Role-based access, hashed passwords
* **API Design:** RESTful endpoints
